# Tic-Tac-Toe Game :
A classic, interactive Tic-Tac-Toe (Noughts and Crosses) web application built with modern web technologies. This project features a clean UI, smooth transitions, and a fully responsive design.

# Features :
Two-Player Gameplay: Play locally with a friend as 'X' and 'O'.

Dynamic UI: Built with HTML5 and CSS3 for a sleek, modern look.

Game Logic: Robust JavaScript logic to handle win patterns and draw scenarios.

Responsive Design: Fully playable on desktops, tablets, and mobile devices.

Reset Functionality: Quickly restart the game at any time with a single click.

# Technologies Used :
HTML5: For the structure of the game board.

CSS3: For styling, grid layout, and animations.

JavaScript (ES6): For game mechanics, win detection, and DOM manipulation.

# How to Play :
The game is played on a grid that's 3 squares by 3 squares.

Player 1 is X, and Player 2 is O. Players take turns putting their marks in empty squares.

The first player to get 3 of their marks in a row (up, down, across, or diagonally) is the winner.

When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.
